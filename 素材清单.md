## 三、产品图片（核心优先级最高）

### 规格要求（所有产品图统一标准）
- **尺寸**：1200×1200px（正方形）
- **背景**：纯白（#FFFFFF）
- **格式**：JPG 或 WebP
- **大小**：单张 ≤ 300KB（WebP 压缩比更好，推荐）
- **命名**：`{产品编号}-{序号}.jpg`，例如 `SS-1525-1.jpg`、`SS-1525-2.jpg`

### 每个产品需要的图片
| 序号 | 内容 | 必须？ |
|------|------|--------|
| -1 | 正面白底主图 | 🔴 必须 |
| -2 | 45° 侧面图 | 🔴 必须 |
| -3 | 顶部/盖子细节 | 🟡 重要 |
| -4 | 生活场景图（手持/桌面） | 🟡 重要 |
| -5 | 定制 Logo 效果图 | 🟢 可后补 |

> **上线优先级**：先把主推产品的 -1 和 -2 备齐，其余陆续补充。

## 九、上传到 Payload 的操作说明

1. 打开 `http://localhost:3000/admin`
2. 左侧菜单 → **Media** → 右上角 **Create New**
3. 上传图片后 Payload 自动生成 3 个尺寸（thumbnail/card/full）
4. 产品图片上传完成后，再去 **Products** 里创建产品并关联图片

## 十、图片处理建议

- 批量压缩工具：[Squoosh](https://squoosh.app/)（免费，支持 WebP）或 [TinyPNG](https://tinypng.com/)
- 白底抠图：[remove.bg](https://www.remove.bg/)（每月 50 张免费）
- 图片尺寸调整：macOS 预览 → 工具 → 调整大小
